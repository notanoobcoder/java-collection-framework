# java-collection-framework
This repo contains basic codes needed to understand the most-used collection framework classes in java.

They include:

- ArrayList()
- LinkedList()
- HashSet()
- LinkedHashSet()
- HashMap<K, V>()
- LinkedHashMap<K, V>()
- TreeMap<K, V>()
